Fala pessoal, passei por algumas raivas ao usar o SDK manager no windows 7, algumas vezes eu o instalava e ele funcionava e outras não (a maioria). O pior erro é o que ele não encontrava a JAVA_HOME, mesmo ela já estando dentro das variáveis de ambiente do sistema. 
Assim toda vez que executava o antroid.bat pelo prompt recebia a mensagem:
ERROR: No suitable Java found. In order to properly use the Android Developer
Tools, you need a suitable version of Java JDK installed on your system.
We recommend that you install the JDK version of JavaSE, available here:
  http://www.oracle.com/technetwork/java/javase/downloads




pesquisei pesquisei e achei esta solução no stackoverflow:

This is how you fix it:
  1. Open up tools/android.bat in your favorite text editor
  2. Search for this piece of code:
    set java_exe=
    call lib\find_java    .bat
    if not defined java_exe goto :EOF
  3. Replace it with this:
    set java_exe="D:\Program Files\Java\jdk1.7.0_07\bin\java.exe"
    where the path is the path to your Java exe.
  4. Run android.bat

IMPORTANTE:

  1. Faça um backup do arquivo android.bat
  2. O arquivo android.bat fica dentro da pasta "tools" da sua instalação do SDK
  3. No meu caso, a dica acima só funcionou sem as aspas. :D
  4. caso ainda assim nao funcione pra você execute o antroid.bat pelo prompt, pois assim será possível ver o erro que esta dando.
