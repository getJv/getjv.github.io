Fala povo,

Anteriormente postei a solução para quando ao executar o sdk manager ele não encontrava o diretório onde estava instalado o java. Mas depois deste, veio outro problema hahaha, normal na TI né!

Desta vez é o SDK não encontra o swt.jar e reclama que não existe a variável de ambiente ANDROID_SWT. Pois bem! procurei procurei e achei a solução.



Para resolver este problema siga estes passos:

  1. Baixe a swt.jar (64bits) neste endereço; ( para outras versoes ou SO acesse aqui )
  2. crie um diretorio "android_swt" em um local preferido, preferencialmente um endereço em espaços no nome, tais como "C:\android_swt" ou "D:\android_swt"
  3. descompacte o conteudo arquivo zip dentro do diretorio criado.
  4. anote o endereço do diretorio e crire uma variavel de ambiente com o nome: "ANDROID_SWT" e no valor coloque o caminho da pasta com o arquivo swt.jar
  5. apra um propt de comando e acesse o o local onde vc instalou a SDK android acesse a pasta tools e execute o arquivo android.bat.
  6. aguarde alguns segundos e o sdk manager vai abrir. 
  7. Caso não abra... o log com o motivo do erro vai aparecer no prompt.
É isso galera. espero que ajude.
