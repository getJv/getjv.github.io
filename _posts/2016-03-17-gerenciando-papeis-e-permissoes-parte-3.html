Hoje continuaremos a falar sobre o gerenciamento de papéis e suas permissões. Caso queiram ver o início dessa série basta consultar a parte 1 deste artigo, ou a parte 2.

Nesse artigo continuaremos a ver cada uma das permissões de cada módulo e também um exemplo de acesso a cada nova opção que é liberada ao usuário quando tal permissão é atribuída ao seu papel.


Espero que gostem e Bons estudos!



Agenda deste artigo

Este é o terceiro artigo da série de Papéis e Permissões, nele você vai encontrar:

  • Um descrição das permissões dos módulos

    • e Noticias
  • Para a explicação das demais permissões consulte a artigo 3º desta série.

Então sem mais delongas vamos as permissões.

Módulo: Fóruns
  • Gerenciar fóruns: Permite criar, editar e ordenar, abrir e ver os fóruns, somente para os projetos aos quais o usuário/papel esta vinculado. Entretanto não liberar a permissão para acrescentar mensagens ao forum.
      
    Acesso a opção 'Gerenciar Fóruns' no Redmine
  • Postar mensagens: Permite postar mensagem em qualquer fórum dos projetos em que o papel participa.
    Acesso a opção 'Postar Mensagens em Fóruns' no Redmine

  • Editar mensagens: Permite editar mensagens em qualquer fórum dos projetos que o papel participa.
    • ATENÇÃO: Essa permissão permite INCLUSIVE a edição de mensagens onde o autor não foi o próprio usuário, também será possível editar a própria descrição do fórum. Use-a com muita cautela.
      Acesso a opção 'Editar Mensagens em Fóruns' no Redmine

  • Editar próprias mensagens: Permite editar APENAS as mensagens em que o autor é o mesmo usuário da edição. A imagem de exemplo é a mesma utilizada acima, com exceção apenas para a opção de edição da descrição do forúm e de mensagens de outros usuários.

  • Excluir mensagens: Permite excluir mensagens em qualquer fórum dos projetos que o papel participa.
    • ATENÇÃO: Essa permissão permite INCLUSIVE a exclusão de mensagens onde o autor não foi o próprio usuário ou até mesmo o próprio fórum. Use-a com muita cautela.
      Acesso a opção 'Excluir Mensagens em Fóruns' no Redmine

  • Excluir próprias mensagens: Permite excluir APENAS as mensagens em que o autor é o mesmo usuário da edição. A imagem de exemplo é a mesma utilizada acima, com exceção apenas para a opção de exclusão do forúm e de mensagens de outros usuários.

Módulo: Calendário

  • Ver calendário: Permite o acesso ao calendário do projeto onde é possível ver as ações e entregas agendadas de cada tarefa do projeto.
Acesso a opção 'Calendário' no Redmine

Módulo: Documentos


  • Adicionar documentos: Permite adicionar documentos que pertencem ao projeto como um todo (independe de tarefa). Para utilizar esta opção o usuário deve ter também a opção "Ver Documentos".

Acesso a opção 'Documento' no Redmine

  • Editar documentos: Permite editar um documentos já existentes no projeto em que o usuário participa.
  • Excluir documentos: Permite excluir documentos que pertencem ao projeto ao qual o usuário está vinculado.
  • Ver documentos: Permite acessar os documentos de um projeto ao qual o usuário esteja vinculado.

Módulo: Arquivos
  • Gerenciar arquivos: Permite gerenciar (Adicionar, Exluir) os arquivos, diferente de documentos os arquivos são vinculados conforme a versão selecionada para os mesmos. Esta permissão depende da permissão "Ver Arquivos" para poder funcionar

    Acesso a opção 'Arquivos' no Redmine

  • Ver arquivos: Permite acessar a seção de arquivos do projeto.
Módulo Gantt
  • Ver gráfico gantt: Permite que o ator tenha acesso ao gráfico que apresenta visualmente as informações sobre as atividades do sistema.

    Acesso a opção 'Gráfico de Gantt' no Redmine

Módulo Notícias

  • Gerenciar notícias: Permite Gerenciar (Adicionar, Exluir) as notícias sobre o que acontece ao projeto. Seria como um blog para o projeto onde os usuários tomam conhecimento sobrenovas informações e podem comentar elas.

    Acesso a opção 'Notícias' no Redmine
  • Comentar notícias: Permite ao usuário registrar um comentário a alguma notícia que esteja publicada em um de seus projetos.
