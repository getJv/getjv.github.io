Módulo Gerenciamento de Tarefas
  • Gerenciar categorias de tarefas: As categorias são como etiquetas que possibilitam a realização uma segunda organização pára as tarefas de um projeto. Com essa permissão é possível criar, edita ou excluir uma categoria de tarefa.

    Acesso a opção 'Gerenciar Categorias' no Redmine
  • Ver tarefas: E bem simples e permite ao usuário ver as tarefas de um projeto em 'Modo leitura', seu acesso se dá pela aba "Tarefas".
  • Adicionar tarefas: Permite ao usuário adicionar tarefas a um de seus projetos. O acesso se dá pela aba "Nova Tarefa".
  • Editar tarefas: Permite ao usuário editar as tarefas de seus projetos. O acesso se dá pela opção "Editar" dentro da visualização de uma tarefa.

    Acesso a opção 'Editar Tarefa' no Redmine
  • Copy issues: Permite ao usuário criar uma nova tarefa para seus projetos copiando os atributos atuais de uma determinada tarefa.  O acesso se dá pela opção "Copiar" dentro da visualização de uma tarefa. Mas esta função somente funcionará se o ator também tiver a permissão "Adicionar Tarefas". (vide imagem acima).
  • Gerenciar relacionamentos de tarefas: Permite ao usuário relacionar as tarefas existentes em um projeto registrando qual é o tipo de sua relação. Está funcionalidade é muito útil para criarmos um diagrama de dependências do nosso plano de trabalho, mais a frente teremos um artigo apenas para falar sobre isso.  O acesso a funcionalidade se dá pela seção "Tarefas relacionadas" dentro da visualização de uma tarefa.

    Acesso a opção 'Tarefas Relacionadas' no Redmine
  • Gerenciar subtarefas: Permite ao usuário criar tarefas 'Filhas' de uma determinada tarefa. Está funcionalidade é muito útil para subdividir em partes menores uma tarefa que seja muito grande.  O acesso a funcionalidade se dá pela seção "subtarefas" dentro da visualização de uma tarefa.
  • Alterar tarefas para públicas ou privadas: Permite ao usuário alterar a visibilidade de pública para privada e vice-versa para qualquer tarefa de um de seus projeto. Está funcionalidade é muito útil para impedir que em um projeto público  determinadas tarefas seja visíveis as pessoas externas ao projeto.  O acesso a funcionalidade se dá na tela de cadastro ou edição de uma tarefa.

    Acesso a opção 'de visibilidade das tarefas' no Redmine
  • Alterar as próprias tarefas para públicas ou privadas: Permite a mesma ação descrita na permissão acima, entretanto o usuário tem acesso a visibilidade apenas de suas próprias tarefas.
  • Adicionar notas: Permite ao usuário apenas adicionar notas a qualquer tarefa de seus projetos. O acesso a funcionalidade se dá na tela de edição de uma tarefa.
  • Editar notas: Permite ao usuário apenas editar notas em qualquer tarefa de seus projetos. O acesso a funcionalidade se dá na tela de visualização de uma tarefa.
  • Editar suas próprias notas: Permite ao usuário apenas editar suas própria notas. O acesso a funcionalidade se dá na tela de edição de uma tarefa.
  • Ver notas privadas: Permite ao usuário visualizar as notas privadas das tarefas de seus projetos. O acesso a funcionalidade se dá na tela de visualização de uma tarefa.
  • Permitir alterar notas para privada: Permite ao usuário editar as notas privadas das tarefas de seus projetos. O acesso a funcionalidade se dá na tela de edição de uma tarefa.
  • Excluir tarefas: Permite ao usuário excluir tarefas de seus projetos. O acesso a funcionalidade se dá na tela de edição de uma tarefa.
  • Gerenciar consultas públicas: Permite ao usuário criar consultas que sejam públicas a todos os usuários ou até mesmo a somente usuarios de um determinado papel. O acesso a funcionalidade se dá na tela de consulta de tarefas.

    Acesso a opção 'Consulta de tarefas' no Redmine
  • Salvar consultas: Permite ao usuário salvar suas próprias consultas personalizadas, esta serão visiveis somente a quem as salvou.
  • Ver lista de observadores: Permite ao usuário visualizar quem são as pessoas que estão observando determinada tarefa.

    Acesso a opção 'Observadores de tarefas' no Redmine
  • Adicionar observadores: Permite ao usuário adicionar novos observadores a uma determinada tarefa. Esta opção esta disponível no cadastro ou edição de uma tarefa

    Acesso a opção 'Adicionar Observadores de tarefas' no Redmine
  • Excluir observadores: Permite ao usuário excluir quem está observando a tarefa.
  • Import issues: Permite ao usuário realizar a importação de novas tarefas a partir de um arquivo .csv. Esta funcionalidade será estudada em detalhes mais a frente em um artigo específico. O acesso a esta opção se dá pelo caminho: "Projeto >> Tarefas >> Import"

    Acesso a opção 'Importar tarefas via planilha' no Redmine


Módulo Gerenciamento de tempo

  • Adicionar tempo gasto: Permite ao usuário adicionar a quantidade de tempo gasto em uma determinada tarefa. O acesso a esta opção se dá pelo caminho: 'Projeto >> aba tarefas >> tarefa >> opção 'Tempo de Trabalho'

    Acesso a opção 'Adicionar tempo gasto na tarefa' no Redmine
  • Ver tempo gasto: Permite ao usuário ver e acessar a quantidade de tempo gasto e também acessar o relatório de descritivo de horas do projeto. O acesso a esta opção se dá pelo caminho: 'Projeto >> aba Visão geral >> seção 'Tempo Gasto'
    • Acesso a opção 'Tempo Gasto do projeto' no Redmine
  • Editar tempo gasto: Permite ao usuário editar a quantidade de tempo gasto em uma determinada tarefa, independente de quem o tenha cadastrado. O acesso a esta opção se dá pelo caminho: 'Projeto >> aba Visão geral >> seção 'Tempo Gasto' >> Opção 'Detalhes' >> icone editar

    Acesso a opção 'Editar tempo gasto' no Redmine
  • Editar o próprio tempo de trabalho: Permite ao usuário editar a quantidade de tempo gasto em uma determinada tarefa. Mas somente é possivel editar  as horas que foram registradas pelo proprio usuário. O acesso a esta opção se dá pelo caminho: 'Projeto >> aba Visão geral >> seção 'Tempo Gasto' >> Opção 'Detalhes' >> icone editar
  • Gerenciar atividades do projeto: Permite ao usuário gerir as atividades que são utilizadas para o registro de horas do projeto. O acesso a esta opção se dá pelo caminho: 'Projeto >> aba Configurações >> aba 'Atividades'.

    Acesso a opção 'Gerir atividades do Projeto' no Redmine

Os Módulos Gerenciar repositório e Gerenciar Wiki

Dentro do módulo de repositórios temos as permissões: 
  • Pesquisar repositório
  • Ver conjunto de alterações
  • Acesso do commit
  • Gerenciar tarefas relacionadas
e no Módulo Wiki, temos:
  • Gerenciar wiki: 
  • Renomear páginas wiki
  • Excluir páginas wiki
  • Ver wiki
  • Exportar páginas wiki
  • Ver histórico do wiki
  • Editar páginas wiki
  • Excluir anexos
  • Proteger páginas wiki
Sobre estas permissões não colocarei muitas informações aqui pois vamos ter artigos dedicados especialmente para falar sobre cada uma delas.
