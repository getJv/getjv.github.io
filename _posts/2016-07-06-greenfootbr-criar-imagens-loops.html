A equipe do GreenfootBR terminou de legendar mais uma vídeo aula sobre programação de jogos em Java com o GreenFoot. Hoje publicamos Brincando com Códigos Vídeo #32 - Belas Imagens e loops com while

Neste episódio o professor Michael mostrar mais exercícios de fixação para uso de repetições (loops). Durante a revisão ele aproveitará para mostrar como usar os loops para criar padrões de imagens que proporcionarão um designer bacana ao seu jogo. 

Assista, curta e compartilhe essa grande iniciativa!
