# Vue Qlik Components

# The Vue Qdt-components

The Vue Qdt-components it is a Vue library which abstracts all complexity to embeded qlik components inside a webpage using vueJS.

# Setting up the environment

TIP

To know more about please visit:

  • the dedicated qlik community page
  • The github repository
  • The npm repository
  • We will use the version "qdt-components": "^1.1.51", the newer version does not allow to handle the QtdViz.
  1. Inside your project run npm install --save qdt-components@1.1.51 o install the dependency
  2. now put the bellow code into your /components/QlikInterface.vue
    <template>
  <div>
    <p>QdtComponent</p>
    <p>Type: {{ type }}</p>
    <p>Props: {{ props }}</p>
  </div>
</template>

    <script>
//https://github.com/qlik-demo-team/qdt-vue-template/blob/master/src/components/QdtComponent.vue
import QdtComponents from "qdt-components";
const options = {
  config: {
    host: "sense-demo.qlik.com",
    secure: true,
    port: 443,
    prefix: "",
    appId: "133dab5d-8f56-4d40-b3e0-a6b401391bde",
  },
  connections: {
    vizApi: true,
    engineApi: true,
  },
};
const qdtComponents = new QdtComponents(options.config, options.connections);
export default {
  name: "QlikInterface",
  props: {
    type: String,
    props: Object,
  },
  data() {
    return {
      qdtComponents: null,
      qdtViz: null,
      qdtElement: null,
      qAppPromise: null,
      qDocPromise: null,
    };
  },
  mounted() {
    this.getQtdComponent();
  },
  methods: {
    /** Prints all know elements */
    showInnerProps() {
      console.log(this.qAppPromise);
      console.log(this.qDocPromise);
      console.log(this.qdtViz);
      console.log(this.qdtElement);
    },
    /**
     * Start the qdt Elements
     */
    getQtdComponent() {
      this.qdtComponents = qdtComponents;

      if (this.qdtComponents) {
        // render the object
        this.qdtComponents
          .render(this.type, this.props, this.$el)
          .then((res) => {
            this.qdtViz = res;
            this.$emit("update:qdtViz", res);
            res.qVizPromise.then((qviz) => {
              this.qdtElement = qviz;
              this.$emit("update:qdtElement", qviz);
            });
          });

        this.qdtComponents.qAppPromise
          .then((res) => {
            this.qAppPromise = res;
            this.$emit("update:qAppPromise", res);
          })
          .catch((err) => console.log(err));
        this.qdtComponents.qDocPromise.then((res) => {
          this.qDocPromise = res;
          this.$emit("update:qDocPromise", res);
        });
      }
    },
    //Sample to update the title
    updateTitle(newValue) {
      this.qdtElement.model.layout.title = newValue;
    },
    // Function to export data
    exportByTabSeparator() {
      this.qdtElement
        .exportData({
          format: "CSV_T",
          state: "A",
        })
        .then(function(link) {
          window.open(link);
        })
        .catch((err) => console.log(err));
    },
    // Function to export data
    exportByCommaSeparator() {
      this.qdtElement
        .exportData({ format: "CSV_C", state: "A" })
        .then(function(link) {
          window.open(link);
        })
        .catch((err) => console.log(err));
    },
    // Function to export data
    exportToExcel() {
      this.qdtElement
        .exportData({ format: "OOXML", state: "A" })
        .then(function(link) {
          window.open(link);
        })
        .catch((err) => console.log(err));
    },
    // Function to export data
    exportToPdf() {
      this.qdtElement
        .exportPdf({
          documentSize: "a4",
          aspectRatio: 2,
          orientation: "landscape",
        })
        .then(function(link) {
          window.open(link);
        })
        .catch((err) => console.log(err));
    },
  },
};
</script>
    1. Now the project are enable to import qlik components.

    # Engine e viz and inner elements

    the interface emits by vue events the engine e viz elements to your own use. be creative. Or you can use the method showInnerProps to print them at console.

    # Importing elements from qlik

    1. Get the wanted qlik object id from you APP. It can be done by dev-hub or clicking with rigth button over the element.

      1. now create the file: components/EmbedQlikObject.vue with the code:

        <template>
  <div>
    <qlik
      ref="qlikRef"
      :qAppPromise.sync="qAppPromise"
      :qDocPromise.sync="qDocPromise"
      :qdtViz.sync="qdtViz"
      :type="type"
      :props="config"
    />
    <v-btn @click="$refs.qlikRef.exportByCommaSeparator()">
      CVS comma
    </v-btn>
    <v-btn @click="$refs.qlikRef.exportToExcel()"> Excel </v-btn>
    <v-btn @click="$refs.qlikRef.exportToPdf()"> PDF </v-btn>
    <v-btn @click="$refs.qlikRef.exportToPng()"> image </v-btn>
    <v-btn @click="$refs.qlikRef.showInnerProps()"> Inner Props </v-btn>
  </div>
</template>

<script>
import qlik from "./QlikInterface";
export default {
  name: "EmbedQlikObject",
  components: {
    qlik,
  },

  data: () => ({
    qAppPromise: null,
    qDocPromise: null,
    qdtViz: null,
    type: "QdtViz",
    config: {
      id: "MJJA",
      height: "400px",
      exportData: true,
      exportImg: true,
      exportImgOptions: { width: 600, height: 400, format: "JPG" },
      exportPdf: true,
      exportPdfOptions: { documentSize: { width: 300, height: 150 } },
    },
  }),
};
</script>
        1. And its done now you have a full sample about how embed qlik object using vue
